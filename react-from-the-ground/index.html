Server first vs Client first

Ada 2 konsep di Next.js app & pages router

Pages router (client first)

  • SSR (getServerSideProps)
  • SSG (getStaticProps)
  • CSR (useEffect)
  • API routes (pages/api/*)
  • Etc.
Konsep pertama dari Next.js sebelum ada App router.. Jelasin lah per poin

App router (server first)

  • Client component (use client)
  • Server Action (use server)
  • React Server Component (RSC)
  • New routing
    • layout.tsx
    • page.tsx
    • route.ts
  • Partial prerendering (experimental)
  • Etc.
Konsep baru dari Next.js yang dikenalkan sejak versi 13.. Jelasin lah per poin
